## Install & Run

```console
git pull https://github.com/parallel-news/permacast-bloodstone-helper.git

npm install && npm run start

```

## Usage

### Generate RSS Feed For A Channel

```console
GET /feeds/rss/:pid
```

- `pid` : channel ID
